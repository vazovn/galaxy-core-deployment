Accessing the GUI

To access the web gui, open a browser with url: https://mygoldserver/gold.

[gold]# mozilla https://mygoldserver/gold

Note: In order to use the web gui, users will have to generate passwords for themselves using the gchpasswd client command. Gold may have to be restarted in order for role privileges to be reflected in the GUI.

[gold]# gchpasswd
