Deleting Users

To delete a user, use the command grmuser:

grmuser [—debug] [-? | —help] [—man] [—quiet] [-v | —verbose] {[-u] user_name}

Example 6-8. Deleting a user

$ grmuser bob 

Successfully deleted 1 User
