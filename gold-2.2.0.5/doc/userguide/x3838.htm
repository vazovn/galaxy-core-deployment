Adding an Action to an Object

To specify that an action is allowed for an object, use the command goldsh Action Create:

goldsh Action Create Object=<Object Namee> Name=<Action Name> [Display=True|(False)] [Description=<Description>] [ShowUsage:=True]

Example 19-8. Adding a Modify Action to Transaction

$ goldsh Action Create Object=Transaction Name=Modify Description=Modify 

Successfully created 1 Action
