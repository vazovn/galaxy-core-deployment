Querying Jobs

To display job information, use the command glsjob:

glsjob [[-J] job_id_pattern] [-p project_name] [-u user_name] [-m machine_name] [-C queue] [-T type] [—stage stage] [-s start_time] [-e end_time] [—show attribute_name[,attribute_name...]...] [—showHidden] [—raw] [—debug] [-? | —help] [—man] [—quiet] [-V | —version] [[-j] gold_job_id]

Example 13-2. Show specific info about jobs run by amy

$ glsjob —show=JobId,Project,Machine,Charge -u amy 

JobId      Project   Machine Charge 
---------- --------- ------- ------ 
PBS.1234.0 chemistry colony  0
