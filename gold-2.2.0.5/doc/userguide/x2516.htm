Modifying Jobs

It is possible to modify a job record by using the command gchjob:

gchjob [-u user_name] [-p project_name] [-m machine_name] [-o organization] [-C queue_name] [-Q quality_of_service] [-P processors] [-N nodes] [-M memory] [-D disk] [-n job_name] [—applicationapplication] [—executableexecutable] [-t wallclock_duration] [-s start_time] [-e end_time] [-T job_type] [-d description] [-X | —extension property=value...] [—debug] [-? | —help] [—man] [—quiet] [-v | —verbose] [-V | —version] [[-j] gold_job_id | -J job_id]

Example 13-3. Changing a job

$ gchjob -Q HalfPrice --application=NwChem -X Charge=1234 -d "Benchmark" -J PBS.1234.0 

Successfully modified 1 Job
