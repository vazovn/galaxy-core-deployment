Deleting Projects

To delete a project, use the command grmproject:

grmproject [—debug] [-? | —help] [—man] [—quiet] [-v | —verbose] {[-p] project_name}

Example 8-9. Deleting a project

$ grmproject chemistry 

Successfully deleted 1 Project
